Contenu de la distribution


Ce projet se veut auto-suffisant. Tout sera installé dans un seul répertoire (celui où vous aurez décompressé l'archive téléchargé).

Qu'est-ce qui est installé ?

Les fichiers se retrouvent tous sous le répertoire projet :

  • doc : contient toute la documentation
  • lib : contient les API fourni par les tiers
  • bin : contient les fichiers de commandes destinées aux utilisateurs finaux
  • src : contient les fichiers sources (essentiellement les sources java)
  • samples : contient des exemples pour tester les commandes fournies
  • build/classes : contient les classes compilées
  • build/javadoc : contient les fichiers javadoc
  • build/jar : contient le fichier jar du projet
